# Garmin Ecowitt Weather

A Garmin Connect IQ widget that displays live weather data directly from your Ecowitt weather station.

## Features

✓ Wind direction (16 point + degrees)
✓ Wind speed (knots)
✓ Wind gust
✓ Outdoor temperature (°C)
✓ Indoor temperature (°C)
✓ Pressure (hPa)

## Requirements

• Garmin Connect IQ compatible watch
• Ecowitt account
• Application Key
• API Key
• Weather Station MAC address

## Installation

Install from the Garmin Connect IQ Store.

## Configuration

Open Garmin Connect:

Connect IQ Store
→ My Device
→ My Apps
→ Garmin Ecowitt Weather
→ Settings

Enter:

• Application Key
• API Key
• Weather Station MAC
